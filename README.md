### Setup


* Create a bot with the BotFather
* Create a [papertrail](https://papertrailapp.com/dashboard) account
* create a `config.json` file with the following properties:

```
{
    "botToken": "<value>",
    "papertrail": {
        "host": "<value>",
        "port": "<value>"
    }
}
```

* run `npm install`

### Start
`./startbot.sh`
